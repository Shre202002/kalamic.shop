# **App Name**: NexGenShop

## Core Features:

- User Authentication & Authorization: Secure login with OTP via email and phone, JWT-based session management, and role-based access for users and administrators.
- Dynamic Product Catalog: Display products with categories, subcategories, advanced filtering by price, and a debounced search function, including support for product variants (size, color) and basic stock visibility.
- Shopping Cart Management: Functionality to add, remove, and update quantities of items in a shopping cart, with contents persistently stored in the database.
- Secure Checkout & Order Tracking: A streamlined checkout flow that includes address management, a clear order summary, integration with Razorpay for payments, and the ability for users to track their order status.
- Admin Product Control Panel: An administrative interface allowing super admins and regular admins to add, edit, and delete product listings, manage variants, and update stock levels.
- Personalized Wishlist: Users can save products to a personal wishlist, making it easy to track items of interest for future purchases.
- AI-Powered SEO Tool: A generative AI tool assisting administrators in crafting SEO-optimized product descriptions, dynamic metadata, and Open Graph tags based on product details.

## Style Guidelines:

- Primary brand color: A professional and trustworthy muted blue (#456694). This color represents stability and efficiency, chosen to convey a sense of reliability essential for an e-commerce platform. It provides sufficient contrast against light backgrounds.
- Background color: A very light, almost imperceptible cool-grey hue (#F7F9FA). This soft, understated background keeps the focus on product content, reflecting a clean and modern aesthetic. Its subtle blue undertone maintains harmony with the primary color.
- Accent color: A vibrant yet clean light blue-cyan (#4DD6FC). This analogous accent color, while brighter and more saturated than the primary, is used sparingly to draw attention to key interactive elements like calls-to-action, without overpowering the overall design.
- Body and headline font: 'Inter' (sans-serif). This choice ensures excellent readability across all devices, supporting the mobile-first approach. Its modern and neutral aesthetic aligns perfectly with a clean, efficient, and user-focused e-commerce experience.
- Utilize a consistent set of minimal, clean line icons. These icons will be easily recognizable and enhance the mobile-first responsive design, ensuring clarity and intuitive navigation.
- Implement a clean, spacious, and responsive layout that prioritizes mobile devices. Products will be displayed using a card-based system for easy browsing, ensuring optimal content readability and interaction across all screen sizes.
- Incorporate subtle and performant micro-interactions for feedback on user actions (e.g., adding to cart, button clicks). Page transitions should be smooth and fast, enhancing the user experience without sacrificing performance or creating distractions.
